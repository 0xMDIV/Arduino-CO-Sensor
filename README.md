# Arduino-CO-Sensor
Measure the CO in the Air with an Arduino and a MQ-135 Sensor

# Libs used:
https://github.com/GeorgK/MQ135

# Hardware used:
- Breadboard
- Arduino Nano
- MQ-135 Flying Fish
- Buzzer
